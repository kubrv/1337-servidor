### Comandos de administradores
```c
.map NOME_DO_MAPA //ALTERAR MAPA
.fs //STARTAR PARTIDA
.fe //ENCERRAR PARTIDA
.restore NUMERO_DA_RODADA //VOLTAR NA RODADA ESPECIFICADA
!rcon host_workshop_map ID_DO_MAPA //ALTERAR PARA QUALQUERMAPA DA OFICINA
```
